# Tree Farm

We opted for a more simplistic design for a tree farm on this server. The reason behind this decision is that <a href="https://www.youtube.com/@ilmango" target="_blank" rel="noreferrer">Ilmango</a>'s design is resource intensive.

We wanted to take it slow on this SMP world, so we have not gathered many of these resources. Thus, a more simplistic design makes more sense, even with lower yields.

Design: <a href="https://www.youtube.com/@ianxofour" target="_blank" rel="noreferrer">ianxofour</a>.
